# Quickstart: MAST Search Results

To try out the new search features:

1.  **Start the app**:
    ```bash
    mast-tui
    ```

2.  **Search for an object**:
    Type an astronomical object name (e.g., "M31", "Kepler-10", "Eta Carinae") and press `Enter`.
    A status line will show that the search is in progress.

3.  **Navigate results**:
    Use the arrow keys to scroll through rows (`Up`/`Down`) and columns (`Left`/`Right`).
    Notice that the column headers stay at the top while you scroll vertically.

4.  **Reset the view**:
    Type `/clear` to return to the welcome message.

5.  **Exit**:
    Type `/exit` or press `Esc` twice in rapid succession.
