# Quickstart: Advanced Search

## Access the Form
1. Type `/advanced` in the main Command prompt and press `Enter`.
2. The view should switch to the structured form.

## Navigate and Edit
- Use the **Up/Down Arrows** to move the focus (blue highlight) between fields.
- Press **Enter** or **Tab** to start editing the focused field.
- Type your search criteria (e.g., `HST` for Mission).
- Press **Enter** again to save and return to navigation mode.

## Example Search: HST Science Images
1. Navigate to **Mission** and set it to `HST`.
2. Navigate to **Instrument** and set it to `WFC3/IR`.
3. Press **Ctrl+S** (or type `/search`) to execute the search.
4. The view will return to the Results Table showing your filtered data.

## Tips
- Fields support **wildcards** (e.g., `target_name = M8*`).
- To clear the form and start over, type `/clear` within the form.
- Press **Esc** to return to the main dashboard at any time.
